# termbuf

Super basic display driver for terminals.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     termbuf:
       github: asmundstavdahl/termbuf
   ```

2. Run `shards install`

## Usage

```crystal
require "termbuf"
```

See `spec/termbuf_spec.cr` for examples.
